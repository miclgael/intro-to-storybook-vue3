## Quick start

### Clone the repo
```
gh repo clone miclgael/intro-to-storybook-vue3 taskbox

cd taskbox
```
### Install dependencies
```
yarn
```
### Run the test runner (Jest):
```
yarn test:unit
```
### Start the component explorer on port `localhost:6006`:
```
yarn storybook
```
### Run the frontend app proper on `localhost:8080`
```
yarn serve
```
